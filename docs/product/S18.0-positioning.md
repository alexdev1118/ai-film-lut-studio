# S18.0 产品定位

AI Film LUT Studio 是一个桌面优先、本地处理的创意 LUT 工作台。它帮助不熟悉色彩科学的创作者先确认素材状态，再通过参考图或可追溯原创风格生成、比较、预览和导出可验证的 `.cube`，并给出面向剪辑软件的正确使用步骤。

它不是普通 LUT 商店：核心价值不是堆积文件，而是输入边界、风格比较、Preview/Cube 一致性和使用防错。

它不是 DaVinci 替代品：技术还原、镜头级曝光白平衡和局部调色仍应在 NLE 中完成。

它不是专业胶片模拟插件：当前风格是原创参数设计或公开视觉特征启发，不宣称官方胶片测量复刻。

它不把启发式本地颜色统计宣传为机器学习模型，也不把未经核验的相机目录宣传为官方技术转换。
