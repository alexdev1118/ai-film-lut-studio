# S18.0 用户分层

## 核心用户

无色彩科学基础、但已经开始拍摄照片或视频的个人创作者。他们需要快速得到可复现的电影感，同时最容易混淆 Log 还原、创意 LUT 和软件加载位置。

## 次要用户

- 使用无反相机 Log 的个人视频创作者：需要相机来源记录和明确的 DaVinci 技术还原步骤。
- 摄影与视频混合创作者：需要同一风格在静帧预览和视频软件中保持方向一致。
- 小型商业与新媒体团队：需要可追溯、可重复下载、可交接的 LUT 文件和使用说明。
- 有基础的 DaVinci / Premiere 用户：需要专业契约、Hash、Round-Trip 和相机证据，但不应让这些信息干扰快速路径。

## 暂不作为核心用户

- 专业电影调色师：当前产品不能替代完整调色台、ACES 管线或专业胶片模拟插件。
- 需要云端批量视频处理的团队：本阶段不处理整条视频，也没有后端。
- 需要官方精确相机技术转换的制作：当前 verified technical transforms 为 0。
- 手机端重度编辑用户：桌面优先，手机仅保持不崩溃和数据安全。

## 控制复杂度

核心用户默认只需要回答素材状态、风格来源、强度和目标软件。次要专业用户可以切换专业模式查看输入 Profile、Gamma、Gamut、Range、点数、Hash、Round-Trip 与 CAMMON。
