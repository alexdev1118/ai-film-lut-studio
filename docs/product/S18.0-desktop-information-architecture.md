# S18.0 桌面信息架构

## 快速模式

1. 素材状态：灰色 Log、正常颜色、不确定。
2. 风格来源：参考图片、风格库、手动调节。
3. 强度：自然 35%、标准 50%、浓郁 70%、完整 100%。
4. 目标软件：DaVinci、Premiere、Final Cut、相机监看、其他 Cube 软件。
5. 生成预览。
6. 导出与使用步骤。

快速模式隐藏 Hash、MAE、Round-Trip、技术证据详情和相机监看高级配置，但不删除这些状态。

## 专业模式

保留输入 Profile、Gamma、Gamut、Range、17/33/65、结构化参数、Hash、契约、Round-Trip、技术转换证据和 CAMMON。专业模式与快速模式共用 WorkspaceProvider、preparedPostLut、RGB Core 和下载 Artifact。

## 风格库

搜索和分类仍是入口；新增 2–4 个候选的同图比较、统一强度和一键带入。详情页继续承担来源、许可、风险和六场景压力预览。

## 导出中心

普通层显示文件名、点数、输入输出假设、目标软件加载位置和常见错误。验证详情折叠显示参数 Hash、Cube Hash、parser、validator 和 Round-Trip 信息。
