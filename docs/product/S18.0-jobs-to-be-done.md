# S18.0 Jobs To Be Done

## 主要任务

当我有一张正常显示的静帧或一段灰色 Log 素材时，我想在不知道完整色彩科学的情况下选择合适的风格并导出正确用途的 LUT，这样我能在 DaVinci、Premiere 或 Final Cut 中复现网站预览，而不会把技术转换和创意风格弄混。

## 支撑任务

1. 判断当前静帧能否直接做创意匹配。
2. 用参考图片或可信原创风格获得方向。
3. 在同一素材上比较候选，不需要打开十二个详情页。
4. 用自然、标准、浓郁、完整四档理解强度。
5. 下载一个经过 parser、validator 和 Hash 验证的 Cube。
6. 得到面向目标软件的明确加载位置和顺序。
7. 出现不一致时再进入专业诊断，而不是首次任务就学习 Round-Trip。

## 成功标准

- 上传到导出不超过七个主要步骤。
- 用户知道 POST LUT 不包含 Log 技术转换。
- 同参数的快速与专业模式导出相同 Cube Hash。
- Preview 与下载 Artifact 来自同一 preparedPostLut。
