# S18.0 路线图

## 当前阶段

- 桌面快速/专业模式。
- 素材状态防错。
- 四风格同图比较。
- 目标软件使用指南。
- 确定性示例项目和下载 Artifact。

## 下一阶段候选

1. 用真实可招募用户执行可用性测试，不以模拟结果代替访谈。
2. 将“原始拍摄来源”和“当前静帧像素状态”建成两个独立但关联的表单段。
3. 在导出结果页加入可复制的软件步骤和输入检查清单。
4. 对合法自有素材建立更接近真实创作的风格比较预览。
5. 恢复 GitHub 凭据后推送检查点分支，进行代码评审。

## 延后

手机端完整体验、整条视频、云端 AI、更多相机品牌、更多风格、正式插件和厂商 LUT 分发全部延后，直到桌面 Beta 证据支持。
