# S18.0 Competitor Matrix

Retrieved: 2026-07-16

This matrix uses official product, documentation, support, standards, and upstream repository pages. Vendor performance claims are recorded as claims, not independent test results. Exact prices are included only when the official page exposed them at retrieval time.

## Coverage

- 21 current products, ecosystems, standards, or open-source projects.
- Commercial film emulation: Dehancer, FilmConvert Nitrate, ARRI Film Lab.
- Camera matching and automation: CineMatch, Colourlab AI, Retouch4me Color Match.
- Web LUT creation and libraries: fylm.ai, Lutify.me, 3D LUT Creator.
- NLE workflows: DaVinci Resolve, Premiere Pro Lumetri, Final Cut Pro.
- Camera look ecosystems: Panasonic REAL TIME LUT, ARRI Look Library and its browsing app.
- Open color infrastructure: OpenColorIO, Config ACES, Colour, OpenImageIO, ACES, LUTCalc.

The full field-by-field matrix is in `S18.0-competitor-matrix.json`.

## Evidence-Backed Patterns

1. **Successful beginner products ask about the source before the look.** FilmConvert and CineMatch center camera/profile selection; Final Cut separates camera LUTs from custom creative LUTs.
2. **Professional products separate technical transforms, creative decisions, and display transforms.** fylm.ai names IDT/LMT/ODT, ACES defines input/look/output transforms, and ARRI versions LogC3 and LogC4 assets independently.
3. **The market sells speed, not only looks.** CineMatch sells faster camera matching, Colourlab sells batch/shot matching, and fylm.ai offers a guided Magic Mode.
4. **Visual selection matters more than a larger catalog.** ARRI exposes 87 looks through groups and intensity previews; DaVinci provides wipe, split-screen, and gallery comparison.
5. **Export without a contract is unsafe.** Official Apple, Adobe, fylm.ai, Lutify.me, ACES, and ARRI documentation all make input/output position material to correct results.
6. **Local processing is a real differentiator.** Colourlab explicitly promotes on-device work; desktop plugins and open-source libraries keep media local by default. Cloud-first web tools must earn trust.
7. **A complete NLE is not the opportunity.** DaVinci already provides deep nodes, scopes, tracking, and color management for free. The product opportunity is to make a correct, portable look easier to create and use.

## Product Gaps Worth Pursuing

- A desktop-first beginner path that asks observable questions instead of starting with Gamma/Gamut vocabulary.
- A clear declaration of what the exported LUT contains and does not contain.
- Side-by-side style selection on the user’s own frame, with risk notes for skin, red, sky, highlights, and shadows.
- A target-software export guide that tells users where the LUT goes and prevents double transforms.
- A local-first reference-match workflow with transparent heuristic confidence, not a black-box professional claim.

## Practices Not To Copy

- Large preset counts without a comparison workflow.
- One label, “LUT,” for technical conversion, creative look, and display transform.
- Expert diagnostics in the default path.
- Camera-support claims without versioned evidence.
- Cloud upload as the only way to test a personal frame.
